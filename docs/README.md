# Documentation Index

This folder keeps non-runtime project material out of the repository root.

## Folders

- `project/` - product notes, user stories, final status, technical overview, and historical project documents.
- `auth/` - authentication and account-flow planning.
- `database/` - database and migration notes.
- `git/` - Git workflow and repository safety notes.
- `generated/` - rendered PDFs, screenshots, diagrams, and generated documentation assets.
- `presentations/` - slide decks and presentation files.
- `tools/` - scripts used to generate or review documentation artifacts.

## Root Files That Stay At Repository Root

The following files intentionally stay at root because build, deploy, or contributor workflows expect them there:

- `RescateAcademico.sln`
- `Dockerfile`
- `railway.toml`
- `global.json`
- `README.md`
- `AGENTS.md`
- `LICENSE.txt`
- `.dockerignore`
- `.gitattributes`
- `.gitignore`
